ViewComponent
Un marco para crear componentes de vista encapsulados, probables y reutilizables en Ruby on Rails.

Documentación
Consulte viewcomponent.org para obtener documentación.

Instalación
En Gemfile, agregue:

gem  "view_component" ,  requiere : "view_component / engine"
